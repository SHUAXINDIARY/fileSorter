# 项目简介 - About the Project 

对指定目录下的所有文件进行分类 - Classifies all files in the specified directory


# 使用 - usage

```shell

# 安装 - install
npm i fs-killer -g

# eg：cd [dir]
cd ~/Desktop/imgs

# 执行 - execute
sorter

```

### 所有选项 - all options

```shell

# specifying a directory
sort -d source

# specifying a file types
sort -t JPG

```
